# Zomato-Data-Analysis
📋 Project Overview
This project involves a comprehensive data analysis of the Zomato restaurant dataset. The objective was to analyze restaurant trends, geographical distribution, pricing strategies, and customer feedback (ratings) to derive actionable business insights.

🛠 Tools & Technologies Used
Excel: Data cleaning, Power Pivot, Data Modeling, and Dashboard visualization.

SQL: Data querying, complex joins, and extracting key performance metrics.

Data Analysis: Trend analysis, payment method distribution, and delivery performance.

📂 Project Structure
Zomato Analysis Report.xlsx: Contains the cleaned data, supporting tables (Country, Currency, Calendar), and the final interactive Dashboard.

Documentation/Analysis_Queries.sql: Contains the SQL scripts used to answer specific business questions (e.g., booking trends, online delivery availability).

📊 Key Findings
Analyzed restaurant distribution across various cities and localities.

Examined the relationship between price ranges, average cost for two, and customer ratings.

Evaluated the adoption of "Online Delivery" and "Table Booking" features across different regions.

Built a dynamic dashboard to visualize restaurant performance metrics.
